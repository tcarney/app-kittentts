# Home Assistant App: KittenTTS

![Supports aarch64 Architecture][aarch64-shield] ![Supports amd64 Architecture][amd64-shield]

Home Assistant app that uses [KittenTTS](https://github.com/KittenML/KittenTTS) (StyleTTS2-based, CPU-only) for text-to-speech via the [Wyoming protocol](https://github.com/OHF-Voice/wyoming).

For installation and configuration, see [DOCS.md](kittentts/DOCS.md).

This app uses the [wyoming-kittentts](https://github.com/tcarney/wyoming-kittentts) server.  It is based on the [piper app](https://github.com/home-assistant/addons/tree/master/piper) implementation.
